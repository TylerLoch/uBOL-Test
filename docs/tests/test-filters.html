uBO Lite Test Page

Test filtering capabilities according to selected filtering mode, basic, optimal, or complete:

  • Except when tagged with @, all tests should fail when no filtering is selected
  • Results are meant to be the same regardless of how the page is navigated to, including at browser launch

Requirements:

  • Version of uBO Lite above 2025.3.2.1298
  • "uBO Lite Test Filters" enabled in the Filter lists pane in uBO Lite’s dashboard
